# Changelog

All notable changes to Versioned Game Saver will be documented in this file.

## [0.1.0] - 2026-06-05

### Added

- Initial public release preparation.
- Windows desktop app for local, versioned game-save snapshots.
- Portable backup library folder selection.
- Bundled game-save manifest scanning for detected games and live save data.
- Manual game profiles.
- Support for tracking save folders, custom folders, and single save files.
- Quick ZIP snapshot creation.
- Snapshot notes with autosave.
- Version restore with automatic safety snapshot creation.
- Separate safety snapshot view.
- Version overwrite and delete actions.
- Save entry and game backup delete actions with confirmation.
- Version aliases, sorting, column reordering, and multi-version delete.
- Background backup and overwrite operations with status feedback.

### Notes

- Release executables are unsigned.
- Official binaries are distributed through GitHub Releases.
- Runtime scanning is offline and uses the bundled sanitized manifest.
